# 📊 프로젝트 기획서: 주식 모멘텀 & 수급 분석 대시보드 (Project "TrendCatcher")

## 1. 프로젝트 개요 (Overview)

* **목표:** 국내/미국 전종목의 일별 시세와 주체별 수급(기관, 외국인, 개인) 데이터를 수집하여, 이제 막 우상향을 시작하는 종목과 시장을 주도하는 섹터를 한눈에 파악할 수 있는 자동화 대시보드 구축.
* **타겟 유저:** 퀀트 기반의 스윙/중장기 투자를 지향하는 개인 투자자 (Steve 본인)
* **핵심 가치:** 엑셀 수기 관리의 비효율성을 제거하고, '가격(차트)'과 '수급(돈의 흐름)'이 동시에 좋아지는 섹터와 종목을 기계적으로 필터링하여 감정을 배제한 투자 인사이트 제공.

## 2. 시스템 아키텍처 및 기술 스택 (Tech Stack)

AI 코파일럿이 코드를 작성할 때 기준이 될 기술 스택입니다.

* **언어:** Python 3.10+
* **데이터 수집 (Data Fetching):**
* 국내(수급+시세): `pykrx` (기관/외국인/개인 순매수 데이터 및 섹터/테마 정보 완벽 지원)
* 미국(시세): `yfinance` 또는 `FinancialModelingPrep API`


* **데이터 전처리 (Processing):** `pandas`, `numpy`, `pandas-ta` (기술적 지표 계산)
* **대시보드 UI (Frontend):** `Streamlit` (파이썬만으로 빠르고 강력한 웹 앱 구축)
* **스케줄링 (Automation):** `GitHub Actions` 또는 로컬 `cron` (매일 장 마감 후 데이터 업데이트)

## 3. 핵심 기능 요구사항 (Requirements)

### 기능 1. 매일 장 마감 후 데이터 파이프라인 자동화 (Data Pipeline)

* **시세 데이터 수집:** 최근 1년 치 전종목 일봉 데이터 (종가, 거래량) 수집.
* **수급 데이터 수집 (국내주식 핵심):** * 매일 기관, 외국인, 개인의 순매수/순매도 대금 수집.
* *로직:* 최근 5거래일 누적 '기관+외국인 쌍끌이 순매수' 종목 필터링.


* **섹터/테마 매핑:** 개별 종목을 산업군 및 테마별로 그룹화(Mapping).

### 기능 2. 우상향 & 수급 스크리닝 알고리즘 (Screening Logic)

* **1차 필터 (차트):** 이동평균선 정배열 초기 진입 종목 (예: 20일선이 60일선을 상향 돌파하는 골든크로스 또는 5>20>60>120 정배열).
* **2차 필터 (수급):** 개인은 매도하고, 기관/외국인의 순매수세가 최근 3~5일간 유입된 종목.
* **3차 필터 (거래량):** 전일 대비 거래량이 200% 이상 급증하며 양봉을 띄운 종목.

### 기능 3. 한눈에 들어오는 직관적인 대시보드 (Dashboard UI)

대시보드는 크게 3가지 뷰(View)로 구성됩니다.

1. **섹터 히트맵 (Sector Heatmap) 뷰:**
* 트리맵(Treemap) 형태로 전 섹터의 당일 평균 등락률 시각화.
* 기관/외국인 자금이 가장 많이 몰린 TOP 3 섹터 하이라이트.


2. **오늘의 발굴 종목 (Top Picks) 뷰:**
* 스크리닝 알고리즘을 통과한 종목 리스트를 테이블 형태로 제공.
* 표에 들어갈 컬럼: `종목명`, `현재가`, `등락률`, `외국인순매수(5일)`, `기관순매수(5일)`, `차트상태(예: 정배열 초기)`.


3. **종목 상세 (Detail) 뷰:**
* 테이블에서 특정 종목 클릭 시, 가격 캔들 차트와 '수급(개인/외국인/기관) 누적 꺾은선 그래프'를 오버레이하여 보여줌.
